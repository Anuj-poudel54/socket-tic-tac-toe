# Running Server

In `server_host.txt` file write `self` or `<eg: localhost>`. Write at the top of the file. Then run `server_socket.py`.

`self` -> Runs in your private IP address.

# Running client

In `addr.txt` file, write the host and port of the server in `<host> <port>` format. Write at the top of the file. Then run `game.py` file.
